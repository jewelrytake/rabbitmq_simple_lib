# rabbitmq_simple_lib
Include in own project with: 
Gradle: compile files('your path/rabMQ.jar')
Maven: 
<dependency>
    <groupId>hire</groupId>
    <artifactId>rabMQ</artifactId>
    <version>1.0-SNAPSHOT</version>
    <scope>system</scope>
    <systemPath>${project.basedir}/rabMQ.jar</systemPath>
</dependency>
